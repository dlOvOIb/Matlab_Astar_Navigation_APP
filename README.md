# Matlab_Astar_Navigation_APP
A simple Matlab APP used Astar

## 1.导航界面
导航界面由坐标转换函数、起点输入、终点输入、导航模式、开始导航、选择地图和地图显示等多个元素组成。
 
<img width="607" height="420" alt="image" src="https://github.com/user-attachments/assets/264c8373-d44a-4ae5-ab9f-c84426d1a00c" />

### 1.1	坐标转换函数
由于校园导航的目的是给出一条教室到教室的路径，所以不能直接输入地图上的坐标来确定起点与终点，此时就需要使用坐标转换函数将不同的教室编号对应到地图上相应的坐标。
自定义坐标转换函数时，要注意输出的结果应为矩阵的行列索引，即一个一行两列的数组。使用时直接将自定义的坐标转换函数的名称输入导航界面的相应位置。

### 1.2 起点、终点的输入
当未添加坐标转换函数时，直接输入起点终点的XY坐标（注意不是行列索引）。添加坐标转换函数时，按自己设置的数据填写即可。

### 1.3 导航模式
由于教学楼依山而建，因此室内道路和室外道路有所不同，室内以楼梯为主，室外以斜坡为主，而从本人体感角度来说，斜坡的劳累程度小于楼梯，但走楼梯的距离要更近。因此出现了距离优先模式（多楼梯、少斜坡，地图中蓝色格子为斜坡、粉色为楼梯）和体力优先模式（多斜坡、少楼梯）两种导航模式。

### 1.4 开始导航
点击开始导航后，会按照设置的起始点及导航模式显示最近的路径，这样就可以快速从一个教室到另一个教室。

### 1.5 选择地图和地图显示
点击选择地图后，会跳出文件窗口让你选择文件，此时只需要选择自己想要放置的地图即可。选择地图后，开始导航按键下方会实时显示当前地图名称。

## 2.导航界面
除导航功能以外，本软件还提供了地图编辑功能。有两种模式，一为新建地图，二为修改原有地图。

### 2.1 新建模式
新建模式需要输入所建地图的行列大小和名称，并且需要选择文件保存路径（默认为app主程序代码所在文件夹中的MAP文件夹）。
<img width="432" height="299" alt="image" src="https://github.com/user-attachments/assets/2767b18c-4167-4421-acf0-3dda8e358acf" />

### 2.2 修改模式
修改模式则是打开已有地图并编辑，需要用户自己选择地图文件。
<img width="468" height="324" alt="image" src="https://github.com/user-attachments/assets/e7b79a1e-29a3-4c35-945e-fb975c0d2ae3" />

### 2.3 地图编辑
输入地图大小和名称后，点击开始编辑后，会自动新建一个地图，并在右侧UI中显示出来。
<img width="469" height="325" alt="image" src="https://github.com/user-attachments/assets/e544a139-a399-4835-a6dd-90f4ff41e9e1" />
我们可以通过点击来编辑地图中每个（除边缘）格子的状态。一个格子有四个状态：空白为正常道路、黑色为墙壁、蓝色为斜坡、粉色为楼梯（顺序为白黑蓝粉循环）。地图数据实时保存。
<img width="469" height="325" alt="image" src="https://github.com/user-attachments/assets/2cad1839-d502-4bcb-b70a-4d302686f707" />
也在地图上添加地点说明，输入要添加说明的位置（XY坐标）及内容，内容实时显示，便于位置调整，点击“添加地点”就可以写入地图中，并且不可调整或删除（注意目前并未实现已经写入的地点说明的调整和删除，添加地点时请三思！！！）。
